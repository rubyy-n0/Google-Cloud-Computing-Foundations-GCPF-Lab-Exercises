# Google Cloud Computing Foundations (GCPF) Lab Portfolio

This repository contains my completed lab exercises for the Google Cloud Computing Foundations (GCPF) program. The course is designed to introduce core cloud computing concepts and provide hands-on experience using Google Cloud Platform (GCP).

## 📘 Overview

The GCPF program covers foundational knowledge and practical skills in cloud computing, including infrastructure, networking, security, data, machine learning, and artificial intelligence. Through this series of labs, I gained hands-on experience in configuring and managing cloud resources on GCP.

## 🧠 Course Modules

The labs are organized according to the official course structure:

* Cloud Computing Fundamentals
* Infrastructure in Google Cloud
* Networking and Security in Google Cloud
* Data, Machine Learning, and AI in Google Cloud

## 🛠️ Skills & Technologies Used

* Google Compute Engine (VMs, load balancing)
* Cloud Storage & Databases
* Virtual Private Cloud (VPC) & Firewall Rules
* Identity and Access Management (IAM)
* Encryption and Security Services
* Big Data and Analytics Tools
* Machine Learning and AI Services on GCP

## 📂 Repository Structure

Each folder contains lab work organized by module. Every lab includes:

* Objective (in my own words)
* Key steps performed
* Tools and services used
* Results and outputs
* Screenshots as evidence

## 📊 Assessment Requirement

This repository was created as part of a **15% Lab Exercise Assessment**. It demonstrates practical understanding and implementation of Google Cloud concepts through 25 completed labs.

## 🚀 Outcome

By completing these labs, I developed a strong foundational understanding of cloud computing and practical experience using Google Cloud services in real-world scenarios.
