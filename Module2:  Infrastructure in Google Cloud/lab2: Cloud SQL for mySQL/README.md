# Cloud SQL for MySQL: Qwik Start

## Objective

To learn how to create and connect to a Cloud SQL for MySQL instance and perform basic database operations.

## Key Steps Performed

1. Created a Cloud SQL for MySQL instance named myinstance.
2. Configured MySQL 8 and generated a root password.
3. Connected to the instance using Cloud Shell and the mysql client.
4. Created a database named guestbook.
5. Created a table called entries.
6. Inserted sample records into the table.
7. Retrieved and verified data using SQL queries.

## Tools/Services Used

* Cloud SQL for MySQL
* Cloud Shell
* MySQL Client
* Google Cloud Console

## Results

Successfully created a Cloud SQL instance, connected to the database, created tables, inserted records, and queried data using SQL commands.
