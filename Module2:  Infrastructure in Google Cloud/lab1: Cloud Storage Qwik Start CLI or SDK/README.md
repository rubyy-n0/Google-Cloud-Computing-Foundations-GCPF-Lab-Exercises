# Cloud Storage: Qwik Start - CLI/SDK

## Objective

To learn how to use Google Cloud Storage and manage storage resources through the command line.

## Key Steps Performed

1. Configured the project region.
2. Created a Cloud Storage bucket using gcloud commands.
3. Uploaded an image file (ada.jpg) to the bucket.
4. Downloaded the object from the bucket.
5. Created a folder within the bucket and copied the object into it.
6. Listed bucket contents and object details.
7. Made an object publicly accessible using ACL permissions.
8. Verified public access through the generated URL.
9. Removed public access permissions.
10. Deleted the object from the bucket.

## Tools/Services Used

* Google Cloud Storage
* Cloud Shell
* gcloud Storage CLI
* Access Control Lists (ACL)

## Results

Successfully created and managed a Cloud Storage bucket, uploaded and organized objects, controlled access permissions, and verified public accessibility of stored content.
