# Introduction to APIs in Google Cloud
## Objective

To understand API architecture, authentication methods, RESTful communication, and how Google Cloud APIs can be used to interact with cloud resources.

## Key Steps Performed

1. Explored the Google API Library.
2. Enabled the Google Fitness API.
3. Created a JSON configuration file.
4. Generated an OAuth 2.0 access token.
5. Configured authentication and authorization for API access.
6. Created a Cloud Storage bucket using the Cloud Storage JSON/REST API.
7. Uploaded an image file to the bucket using REST API requests.
8. Verified bucket and object creation in Cloud Storage.

## Tools/Services Used

* Google API Library
* Cloud Storage API
* Cloud Shell
* OAuth 2.0 Playground
* REST API
* JSON
* curl

## Results

Successfully authenticated with OAuth 2.0, created a Cloud Storage bucket through API requests, and uploaded an object using the Cloud Storage JSON/REST API.
