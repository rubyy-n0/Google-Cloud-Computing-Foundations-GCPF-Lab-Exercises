# App Engine: Qwik Start - Python 

## Objective

To learn how to deploy a Python web application using Google App Engine and understand the basics of serverless application hosting.

## Key Steps Performed

1. Enabled the App Engine Admin API.
2. Cloned the sample Python application repository.
3. Configured a Python virtual environment.
4. Tested the application locally using Flask.
5. Modified the application output from "Hello World!" to "Hello, Cruel World!".
6. Retested the modified application.
7. Deployed the application to Google App Engine.
8. Accessed the deployed application through the generated URL.

## Tools/Services Used

* Google App Engine
* Cloud Shell
* gcloud CLI
* Python
* Flask
* Git

## Results

Successfully deployed a Python Flask application to App Engine. Verified that application changes were reflected after deployment and accessed the application through the App Engine URL.
